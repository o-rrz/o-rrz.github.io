工具說明

這個工具可以生成真碼 / 假碼 / 災難碼，每段代碼可選隨機、固定或自訂。
批量生成，並自動保存自訂郵件/用途到 localStorage。

操作步驟

  1. 選擇郵件、隱私等級、用途、轉發、時間、信任層。
  2. 選擇生成模式：真碼 / 假碼 / 災難碼 / 三層全部生成。
  3. 可新增或移除自訂郵件/用途。
  4. 選擇批量生成數量。
  5. 點擊 生成代碼 按鈕。
  6. 查看結果，真碼綠、假碼黃、災難碼紅。

安全提示

  • 災難碼用於應急交付，不應用於真實帳號。
  • 假碼用於迷惑或測試，不代表真碼內容。
  • localStorage 保存自訂代碼，刷新頁面不丟失。
  • 避免洩露真碼給他人。
